Code for connective classification, as one component in a larger pipeline (under development) for shallow discourse parsing.
External requirements:
- Potsdam Commentary Corpus; standoff XML connectives layer and the syntax layer. Specify the rootfolder in the PCC section of config.ini. Download link: http://angcl.ling.uni-potsdam.de/resources/pcc.html
- DiMLex connective lexicon. Specify location in the dimlex section of config.ini. Download link: https://github.com/discourse-lab/dimlex/blob/master/DimLex.xml
- Stanford parser. Specify locations in the lexparser section of config.ini. Download link: https://nlp.stanford.edu/software/lex-parser.shtml#Download
